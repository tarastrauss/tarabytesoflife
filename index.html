About Me


From Sci Fi to Comp Sci:

I'm an LA based web developer from New Jersey who first learned to code in high school. I took a break from programming to attend USC's School of Cinematic Arts and pursue my dreams of working in Science Fiction. After graduating, I worked as an assistant on Battlestar Galactica and as a Creative Executive at a Elijah Wood's SpectreVision. I rejuvenated my passion for programming this fall at General Assembly's Web Development Immersive and fell in love with coding all over again. I have an unhealthy love for DC superheroes, boardgames and a passion for cooking. Watch out - I can now pun in 4 languages!

Projects

Inform

Inform is an application that uses gamification to encourage users to be informed on both sides of any political issue. The application keeps track of how many articles users read on any given topic, and uses an original algorithm to award points based on bi-partisan interest. As my final project for General Assembly, I put emphasis on my back-end algorithm, APIs and token authentication.


MongoDB Express Angular Node.js CSS

HTML5 Bootstrap Alchemy API Google Civics API


Chef's Hat

Chef's Hat is an app geared towards helping users share their secret family recipes with their friends and family. Emphasis was put on user authentication and authorization - don't let those family secrets get out to the public!


Ruby on Rails PostgreSQL Active Record

CSS HTML5 Bootstrap


ConShareto

ConShareto is a concert-tracking website to help the avid music-lover find and keep track of concerts. This was a group project built by myself and 4 other students from my cohort at General Assembly. My role in the project was primarily working with MongoDB schemas, controllers as well as implementing Google Oauth.


Node.js Express MondgoDB CSS HTML5

Bootstrap Google Oauth Jambase API

2048 Crisis

Help Superman recruit other members of the Justice League and fight Lex Luthor before it's too late! This project combines my passion for DC superheroes and love for puzzle games to make a themed game clone of the modern classic, 2048. This was the first web project I ever made. My emphasis went to flexing my javascript muscles and writing original game logic.
